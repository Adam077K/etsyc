*Set 2026-07-19 by Adam. These are THE references we derive KOL's design tokens and design language from — a tight, intentional 5, each chosen for one specific thing it does well. | Folder | Site | What we take from it |
|--------|------|----------------------|
| [`faire/`](faire/) | https://www.faire.com/ | **Overall style + human feeling** — how a maker/wholesale marketplace feels warm, trustworthy, and human at scale. |

| [`tiktok-shop/`](tiktok-shop/) | https://shop.tiktok.com/us | **The "old" shopping experience** — the video-commerce baseline we react against / improve on (what to keep, what to leave behind). |

## How to read this set
The five are deliberately *different* — each carries a unique quality we want in KOL:
- **faire + kotn** set the emotional temperature (human, warm, trustworthy, story-first).
- **lusion** sets the motion/3D/immersion ceiling.
- **cuberto** sets the modern, clean, video-forward UI polish.
- **tiktok-shop** is the incumbent video-shopping reference — the bar to beat, not to copy.

The design tokens (palette, type, motion, spacing, radius, elevation) and the design language should synthesize the *human feeling* of faire/kotn with the *modern motion + clean polish* of lusion/cuberto — a warmer, more human, more cinematic evolution beyond the tiktok-shop baseline.

## Workflow
1. Founder adds screenshots (committed) + videos (ignored) into each folder.
2. Agent explores the screenshots, writes a per-reference "what to steal / narrative" note.
3. Design-Lead synthesizes the token set + design language from this set → redo `KOL-design-system.md`.
