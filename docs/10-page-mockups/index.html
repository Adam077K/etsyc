How to read these mockups

Each page has a dark spec panel bottom-left tying it to its feature spec, route, build wave, and hard acceptance criteria. The panels are mockup-only — not shipped UI.

Tokens (type scale, spacing, radius, shadows, motion, sunbaked palette) are lifted verbatim from apps/kol/src/app/globals.css. Fonts are system stand-ins for Clash Display / General Sans; film frames are gradient stand-ins for real footage (D12 footage is Phase 7). Maker-world pages use Noor's dark-jewel theme to prove per-maker theming.

Status flags: ⛔ Community is blocked pending founder decision D17. 🔜 Events is roadmap v1.1 per D16-8. Both are designed so nothing gets built by accident.
